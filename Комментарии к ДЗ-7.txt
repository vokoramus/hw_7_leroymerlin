    Комментарии к ДЗ-7

Два варианта сохранения цены в БД: только число (int) или список (число (int), валюта, ед.изм.) [5000, 'Р', 'шт.']. Режимом управляет константа PRICE_FULL_SAVING в модуле settings.py (установить True для сохранения в режиме списка).

Имя папок для фото вытаскивается регуляркой из url. Сделал именно так, т.к. в url продуктов на данном сайте отсутствуют символы, запрещенные для использования в качестве имен директорий 
(именно поэтому нельзя в чистом виде использовать наименование товара из тега h1 на странице товара). Также из url (точнее из имени папки для фото, хотя можно было и со страницы товара) регуляркой вытаскивается артикул и записывается в поле "_id". 
Данные действия выполняет функция get_foldername_article (pipelines.py)


Характеристики: сначала собираю два xpath запроса в одно поле (узнал о такой возможности в оф.документации scrapy https://docs.scrapy.org/en/latest/topics/loaders.html), 
в результате получаем список из 28 элементов: первые 14 - названия хар-к, следующие 14 - значения хар-к. 
Затем через функцию characteristics_to_dict, вызываемую через output_processor=Compose, формируется словарь.

189 товаров (1148 фото) парсил около 35 минут. Возможно, поставил слишком "осторожные" настройки частоты запросов... Отчет о работе сохранил в parsing_statistics.txt
